​ 给定一个字符串，求该字符串的最长回文子串的长度。

暴力匹配

​ 两个循环遍历所有子串，再判断所有子串是否回文，再通过比较获取最长回文子串长度，算法复杂度为

朴素算法

​ 先遍历一遍字符串，对于每个 ，以它作为字符串的中心向字符串两边扩展，在遍历获得子串的同时完成对是否回文的判断，这样就将时间复杂度优化到了 ，需要分别计算奇数串和偶数串

代码实现：

int mmax=0;
for(int i=0;i<len;++i){
    int d1=1,d2=0;
    //奇数串
    for(int j=1;j<std::min(i,len);++j){
        if(str[i-j]!=str[i+j]) return;
        d1+=2;
	}  
    //偶数串
    for(int j=0;j<std::min(i,len-i-1);++j){
        if(str[i-j]!=str[i+1+j]) return;
        d2+=2;
    }
    mmax=std::max(mmax,std::max(d1,d2));
}

std::vector<int> odd;
int id=0,mx=0,mmax=0;
for(int i=0;i<len;++i){
    //回文半径
	int k=(i>mx) ? 1 : std::min(odd[2*id-i]/2,mx-i);
    while(0<=i-k && i+k<len && str[i-k]==str[i+k]) ++k;
    --k;
    odd[i]=2*k+1;
    mmax=std::max(mmax,odd[i]);
    if(i+k>mx){
		id=i;
        mx=i+k;
    }
}

偶数串

偶数部分只需要对奇数的代码做一点修改即可

代码实现：
std::vector<int> even;
int id=0,mx=0,mmax=0;
for(int i=0;i<len;++i){
    //回文半径
	int k=(i>mx) ? 0 : std::min(even[2*id-i+1]/2,mx-i);
    while(0<=i-k && i+1+k<len && str[i-k]==str[i+1+k]) ++k;
    even[i]=2*k;
    mmax=std::max(mmax,even[i]);
    if(i+k>mx){
		id=i;
        mx=i+k;
    }
}

改进

在前面我们都是将奇数串和偶数串分开来算，实际上我们可以通过插入无意义的符号来使二者都统一为奇数串。

对于前面给定的字符串 ，我们插入 个分隔符 #，使字符串变成 #a#b#b#c#b#b#b#c#b#b#b#a#d#，这样所有的子串的长度都为奇数了，最后除以2就是真正子串的长度了。
