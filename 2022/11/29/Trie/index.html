Trie

Posted on 2022-11-29 Edited on 2023-09-05 Word count in article: 915 Reading time ≈ 1 mins.

概念

字典树也叫Trie树、前缀树，是一种针对字符串进行维护的数据结构，而不是算法。

字典树是对字符串的一种存储的方式，这棵树的根节点到某一个目标节点的路径就是一个字符串（单词）。

举个例子，下图的字典树：

该字典树存储的单词为：

a
abc
ba
bca
bca
ca
ca

性质

  • 根节点不包含字符，除根节点外的所有节点都包含一个字符
  • 从根节点到某一个节点，路径上经过的所有字符连接起来，为该节点对应的字符串
  • 每个节点的子节点包含的字符各不相同

应用

  • 计算热词：统计字符串在集合中出现的个数
  • 字典序排序
  • 最长公共前缀（LCP）
  • …

实现

字典树的实现主要有插入和查找

假设该字符串集合由26个小写字母组成，我们定义两个数组

int next[100000][26],cnt; //存放该节点的子节点
int num[100000]; //表示该字符串一共出现多少次，为0表示不存在

插入操作

void insert(std::string str){
    int len=str.length();
	int p=0; //记录当前节点位置
    for(int i=0;i<len;++i){
    	int c=str[i]-'a';
        if(!next[p][c]) next[p][c]=++cnt;
        p=next[p][c];
    }
    ++num[p];
}

查找操作

int find(std::string str){
	int len=str.length();
    int p=0;
    for(int i=0;i<len;++i){
		int c=str[i]-'a';
        if(!next[p][c]) return 0;
        p=next[p][c];
    }
    return num[p];
}
